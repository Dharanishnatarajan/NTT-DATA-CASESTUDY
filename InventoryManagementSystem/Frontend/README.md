# React App - Inventory Management Frontend

## Environment Variables

Create a `.env` file in the root directory with:

```
REACT_APP_API_BASE_URL=https://localhost:7290/api
```

## Installation

```bash
npm install
```

## Running the App

```bash
npm start
```

The app will run on `http://localhost:3000`

## Build for Production

```bash
npm run build
```
